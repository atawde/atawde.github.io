Kioku is an interactive chatbot using OpenAI's 'gpt-4o-audio-preview' model. It is based on audio-text-audio model to retain control on LLM input. Spoken prommpt from microphone is converted to text, using python package 'speech_recognizer' and fed to LLM. Text output from LLM is converted to audio using 'pyttsx3' and fed to speaker.

Personal information about user is pre-fed into LLM as system prompt. It includes name, place of birth, names of siblings, spouse, children, other relatives, places lived, workplaces etc.

In next iteration, multimedia will be fed, like short audio or video clips, pictures. User will then be asked to identify people in the videp/picture or speake in audio.

Some analysis on user's recollection can be then made, like visual memory is strong or events from distant past are not recollected etc.

Github Repo: Tracker

Agentic AI program to conduct mock technical interview.

Using 4 agents to ask questions on Python, Javascript, Data Structures & Algorithms and System Design. Handover agent switches between these agents based on input from interviewee.
At the end 'Analyzer' agent does analysis of the answers given and provides feedback.

Model used is chatgpt.
Using BaseModel to structure the response from model and 'rich' to format the text output on terminal.
